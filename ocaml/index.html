Description

How to Think Like a (Functional) Programmer is an introductory programming textbook based on the OCaml language. It is a modified version of Think Python by Allen Downey. It is intended for newcomers to programming and also those who know some programming but want to learn programming in the function-oriented paradigm, or those who simply want to learn OCaml.

OCaml is a powerful high-level programming language with a large emphasis on speed and efficiency. You can find out more about the language by visiting http://caml.inria.fr

This text leverages the power of the language to teach the reader important concepts in computer science and software design, with an emphasis in experimentation and self-directed learning.
